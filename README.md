# ADHD-AI-Task-Planning-System
This project is designed to support students with ADHD in task planning and time management.

## Features
- Task chunking
- Focus timer (Pomodoro)
- Motivation support
- Task prioritization

## System Design
The system follows:
Input → Rule → Tool → Output
